# Hospital-Readmissions-Reduction-Program-dashboard
Power BI dashboard analyzing CMS Hospital Readmissions Reduction Program data across 3,055 hospitals | Excel · DAX · Data Modeling | Project 
Business Question
Why do some hospitals experience higher patient readmission rates than others?
Using public CMS data, I built an end-to-end analytics solution — from raw data cleaning in Excel to an interactive Power BI executive dashboard — to answer this question across 3,055 hospitals, 6 conditions, and 51 states.
